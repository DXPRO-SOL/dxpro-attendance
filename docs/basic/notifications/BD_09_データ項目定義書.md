# 通知システム - データ項目定義書

---

## 1. 通知（Notification）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 受信者ユーザーID | 参照 | 必須 |
| type | 通知種別 | 文字列 | 必須 |
| title | 通知タイトル | 文字列 | 必須 |
| body | 通知本文 | 文字列 | 任意 |
| link | クリック時の遷移先URL | 文字列 | 任意 |
| isRead | 既読フラグ | 真偽 | 自動 |
| readAt | 既読日時 | Date | 自動 |
| createdAt | 受信日時 | Date | 自動 |
| relatedId | 関連エンティティのID | 文字列 | 任意 |
| relatedModel | 関連エンティティのモデル名 | 文字列 | 任意 |
