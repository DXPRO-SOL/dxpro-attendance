# 通知システム - 画面設計書

---

## 1. 通知ベル・バッジ（ヘッダー）

### 概要
全ページのヘッダーに常時表示される通知ベルアイコン。未読件数をバッジで表示する。

### 表示内容
| 要素 | 説明 |
|------|------|
| ベルアイコン | クリックでドロップダウンを展開 |
| バッジ | 未読通知件数（Socket.IOでリアルタイム更新） |
| ドロップダウン | 最新5件の通知プレビュー（タイトル・日時） |
| 「すべて見る」リンク | 通知一覧画面へ遷移 |

---

## 2. 通知一覧（/notifications）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 「全て既読」ボタン | 一括既読操作 |
| 通知テーブル | 種別・タイトル・本文・受信日時・既読状態 |
| 既読ボタン | 個別通知の既読操作 |
| リンク | 通知に関連するページへのリンク |

### 通知種別
| 種別 | 説明 |
|------|------|
| attendance_approved | 勤怠承認完了 |
| attendance_returned | 勤怠差し戻し |
| leave_approved | 休暇承認 |
| leave_rejected | 休暇却下 |
| goal_submit1 | 目標1次承認依頼 |
| goal_approve1 | 目標1次承認完了 |
| goal_completed | 目標最終承認 |
| payroll_issued | 給与明細発行 |
| workflow_submit | ワークフロー承認依頼 |
| workflow_approved | ワークフロー承認完了 |
| chat_mention | チャットメンション |
