# BD-08. 業務フロー図

各業務の流れを Mermaid flowchart 形式で示す。

---

## 1. 勤怠管理業務フロー

```mermaid
flowchart TD
    A([出社]) --> B[出勤打刻
GPS/手動]
    B --> C[業務]
    C --> D[退勤打刻]
    D --> E[月末: 月次確認]
    E --> F{修正あり?}
    F -->|あり| G[勤怠修正]
    G --> E
    F -->|なし| H[承認申請送信]
    H --> I[管理者が確認]
    I --> J{承認?}
    J -->|承認| K[確定
通知・PDF生成]
    J -->|差し戻し| E
```

---

## 2. 休暇申請業務フロー

```mermaid
flowchart TD
    A([申請開始]) --> B[休暇種別・期間入力]
    B --> C{残高チェック}
    C -->|不足| D[エラー表示]
    D --> B
    C -->|OK| E[申請送信
管理者へ通知]
    E --> F[管理者が確認]
    F --> G{判断}
    G -->|承認| H[残高差し引き
勤怠ステータス更新
メール通知]
    G -->|却下| I[却下通知
メール送信]
```

---

## 3. 目標管理業務フロー

```mermaid
flowchart TD
    A([目標設定]) --> B[下書き作成]
    B --> C{提出?}
    C -->|保存| B
    C -->|提出| D[1次承認待ち
マネージャーへ通知]
    D --> E{マネージャー判断}
    E -->|承認| F[1次承認済み]
    E -->|差し戻し| B
    F --> G[2次承認申請]
    G --> H{管理者判断}
    H -->|承認| I[最終承認完了
通知・メール]
    H -->|差し戻し| B
```

---

## 4. 給与発行業務フロー

```mermaid
flowchart TD
    A([月次締め]) --> B[給与ラン作成]
    B --> C[明細作成
基本給・手当・控除入力]
    C --> D{内容確認}
    D -->|修正| C
    D -->|確定| E[発行操作]
    E --> F[社員が閲覧可能に
メール通知]
    F --> G[社員: PDF/CSVダウンロード]
```

---

## 5. ワークフロー業務フロー

```mermaid
flowchart TD
    A([申請者]) --> B[フォーム選択]
    B --> C[申請内容入力]
    C --> D[申請送信
採番]
    D --> E[ステップ1承認者へ通知]
    E --> F{承認者1判断}
    F -->|承認| G{次ステップあり?}
    F -->|差し戻し| C
    G -->|あり| H[次承認者へ通知]
    H --> I{承認者N判断}
    I -->|承認| G
    I -->|差し戻し| C
    G -->|なし| J[申請完了]
```

## 6. 新入社員受け入れフロー（入社前テスト）

```mermaid
flowchart LR
    A([候補者にURL通知]) --> B[テスト受験
認証不要]
    B --> C[自動採点]
    C --> D[結果保存]
    D --> E[管理者がスコア確認]
    E --> F{合否判断}
    F -->|採用| G[社員登録
アカウント発行]

```
