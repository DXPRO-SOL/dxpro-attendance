# BD-05. CSV・Excel 設計書

---

## 1. 入出力ファイル一覧

| # | 種別 | ファイル名 | 形式 | 操作者 | 方向 |
|---|------|-----------|------|--------|------|
| 1 | 社員一覧 | employees.csv | CSV | 管理者 | 出力 |
| 2 | 給与明細 | {氏名}_給与明細_{年}年{月}月.csv | CSV | 管理者/本人 | 出力 |
| 3 | 監査ログ | audit_log_{日時}.csv | CSV | 管理者 | 出力 |
| 4 | スキルシート | skillsheet_{氏名}.xlsx | Excel | 本人/管理者 | 出力 |
| 5 | スキルシート全員 | skillsheet_all.xlsx | Excel | 管理者 | 出力 |

---

## 2. 社員一覧 CSV

| 列名 | 内容 | 型 |
|------|------|-----|
| 社員番号 | 社員固有ID | 文字列 |
| 氏名 | フルネーム | 文字列 |
| 部署 | 所属部署名 | 文字列 |
| 役職 | 役職名 | 文字列 |
| 入社日 | YYYY-MM-DD形式 | 日付 |
| メール | メールアドレス | 文字列 |
| 電話番号 | 連絡先番号 | 文字列 |

文字コード: UTF-8 BOM付き（Excelでの直接開封対応）

---

## 3. 給与明細 CSV

| 列名 | 内容 |
|------|------|
| 社員番号・氏名・対象月 | 基本情報 |
| 出勤日数・欠勤日数・遅刻回数 | 勤怠集計 |
| 残業時間・深夜時間・休日時間 | 残業集計 |
| 基本給・総支給額・手取り・所得税 | 給与金額 |
| 各手当列・各控除列 | 詳細内訳 |

文字コード: UTF-8 BOM付き

---

## 4. スキルシート Excel（2シート構成）

### Sheet1: 基本情報・スキル・資格

| 区分 | 内容 |
|------|------|
| 基本情報 | 氏名・カナ・生年月日・性別・IT経験年数 |
| スキル | 言語/フレームワーク/DB/インフラ/ツール別にスキル名と習熟度（★1〜5） |
| 資格 | 資格名・取得日 |

### Sheet2: 職務経歴

| 列 | 内容 |
|----|------|
| プロジェクト名・顧客名 | 案件基本情報 |
| 開始月・終了月 | 期間 |
| 役割・説明・技術スタック | 担当内容 |
| 工程（7項目） | 要件定義/基本設計/詳細設計/開発/テスト/運用/管理 |

フォーマット: xlsx（ExcelJS 生成）
文字コード: UTF-8
