# 人事・給与・日報 - 機能設計書

---

## 1. 機能概要
社員情報の管理、給与明細の作成・発行、日次業務報告（日報）の記録・閲覧・AI分析を行う機能群。

---

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| HR-01 | 社員登録 | 管理者 |
| HR-02 | 社員情報編集 | 管理者・本人 |
| HR-03 | 社員一覧・検索 | 全員 |
| HR-04 | 社員写真アップロード | 管理者・本人 |
| HR-05 | 社員CSV出力 | 管理者 |
| HR-06 | 給与明細作成 | 管理者 |
| HR-07 | 給与明細発行 | 管理者 |
| HR-08 | 給与明細閲覧 | 社員（自分）・管理者 |
| HR-09 | 給与明細PDF出力 | 社員・管理者 |
| HR-10 | 給与明細CSV出力 | 社員・管理者 |
| HR-11 | 日報作成 | 社員 |
| HR-12 | 日報一覧閲覧 | 全員 |
| HR-13 | 日報スタンプ・コメント | 全員 |
| HR-14 | 日報AIサマリー | 管理者 |

---

## 3. 処理概要

### HR-07 給与明細発行
1. 管理者が明細のステータスを「発行済み」に変更
2. 対象社員から給与明細が閲覧可能になる
3. 社員へ通知・メール送信（「給与明細が発行されました」）

### HR-14 日報AIサマリー
1. 管理者が期間・社員を選択
2. GET /hr/daily-report/api/stream でSSEストリーミング開始
3. AIが対象日報を分析し要約をリアルタイム出力
4. 「メール送信」操作で任意アドレスへ送付可能
