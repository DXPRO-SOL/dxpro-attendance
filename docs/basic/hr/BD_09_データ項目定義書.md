# 人事・給与・日報 - データ項目定義書

---

## 1. 社員（Employee）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 紐付けユーザーID | 参照 | 必須 |
| employeeNumber | 社員番号 | 文字列 | 必須 |
| nameKanji | 氏名（漢字） | 文字列 | 必須 |
| nameKana | 氏名（カタカナ） | 文字列 | 任意 |
| nameEn | 氏名（英字） | 文字列 | 任意 |
| department | 所属部署 | 文字列 | 必須 |
| position | 役職 | 文字列 | 任意 |
| hireDate | 入社日 | Date | 必須 |
| email | 連絡先メール | 文字列 | 任意 |
| phone | 電話番号 | 文字列 | 任意 |
| orgRole | 組織ロール | 区分 | 任意 |
| photoPath | 写真ファイルパス | 文字列 | 任意 |

---

## 2. 給与明細（PayrollSlip）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| runId | 給与ランID | 参照 | 必須 |
| userId | 社員ユーザーID | 参照 | 必須 |
| year | 対象年 | 数値 | 必須 |
| month | 対象月 | 数値 | 必須 |
| basicSalary | 基本給 | 数値 | 必須 |
| allowances | 手当明細配列 | 配列 | 任意 |
| deductions | 控除明細配列 | 配列 | 任意 |
| totalAllowances | 手当合計（自動計算） | 数値 | 自動 |
| totalDeductions | 控除合計（自動計算） | 数値 | 自動 |
| grossPay | 総支給額（自動計算） | 数値 | 自動 |
| netPay | 差引支給額（自動計算） | 数値 | 自動 |
| status | 明細ステータス | 区分 | 自動 |
| note | 社員コメント | 文字列 | 任意 |

### status コード値
| 値 | 説明 |
|----|------|
| draft | 下書き（社員非表示） |
| issued | 発行済み（社員閲覧可） |
| locked | 確定（変更ロック） |
| paid | 支払済み |

---

## 3. 日報（DailyReport）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 作成者ユーザーID | 参照 | 必須 |
| reportDate | 日報対象日 | Date | 必須 |
| content | 業務内容 | 文字列 | 必須 |
| workHours | 作業時間（時間単位） | 数値 | 任意 |
| tomorrowPlan | 翌日計画 | 文字列 | 任意 |
| attachments | 添付ファイルパス配列 | 配列 | 任意 |
| reactions | スタンプ情報 | 配列 | 自動 |
| comments | コメント配列 | 配列 | 自動 |
