# 人事・給与・日報 - 画面設計書

---

## 1. 社員一覧（/hr）

### 概要
管理者は全社員、一般社員は自分の情報を確認できる一覧画面。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | 部署・氏名検索 |
| 社員テーブル | 社員番号・氏名・部署・役職・入社日・操作ボタン |
| 追加ボタン | 管理者のみ表示 |
| CSV出力ボタン | 管理者のみ表示 |

---

## 2. 社員情報編集（/hr/edit/:id）

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 氏名（漢字） | テキスト | 必須 | 正式氏名 |
| 氏名（カタカナ） | テキスト | 任意 | カタカナ |
| 氏名（英字） | テキスト | 任意 | ローマ字 |
| 社員番号 | テキスト | 必須 | 一意の社員番号 |
| 部署 | テキスト | 必須 | 所属部署 |
| 役職 | テキスト | 任意 | 職位 |
| 入社日 | 日付 | 必須 | 入社年月日 |
| メールアドレス | メール | 任意 | 連絡先メール |
| 電話番号 | テキスト | 任意 | 電話番号 |
| 組織ロール | 選択 | 任意 | admin/manager/team_leader/employee |
| 有給残日数 | 数値 | 任意 | LeaveBalance に保存 |
| 社員写真 | ファイル | 任意 | jpg/png（uploads/daily/に保存） |

---

## 3. 給与明細ダッシュボード（/hr/payroll）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 折れ線グラフ | 直近12ヶ月の月次給与推移 |
| 明細一覧 | 年月・基本給・手当合計・控除合計・差引支給額・ステータス |
| PDF/CSVボタン | 各明細の個別ダウンロード |

---

## 4. 給与明細作成・編集（/hr/payroll/admin/new, edit/:slipId）

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 対象社員 | 選択 | 必須 | 給与明細を作成する社員 |
| 対象年月 | 年月 | 必須 | 給与支給対象月 |
| 基本給 | 数値 | 必須 | 月次基本給与額（円） |
| 各種手当 | 数値（複数） | 任意 | 交通費・住宅手当等 |
| 各種控除 | 数値（複数） | 任意 | 健康保険・厚生年金・所得税等 |
| ステータス | 選択 | 必須 | 下書き/発行済み/確定/支払済み |
| 社員コメント | テキスト | 任意 | 明細に表示するコメント |

---

## 5. 日報作成・一覧（/hr/daily-report）

### 日報作成フォーム項目
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 報告日 | 日付 | 必須 | 日報対象日 |
| 業務内容 | テキストエリア | 必須 | 実施した業務の詳細 |
| 作業時間 | 数値 | 任意 | 業務に費やした時間（時間単位） |
| 翌日計画 | テキストエリア | 任意 | 翌日の予定・計画 |
| 添付ファイル | ファイル | 任意 | 最大10件・20MB/件 |

### 日報一覧
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | 社員・期間（管理者は全社員、一般は自分のみ） |
| 一覧テーブル | 氏名・日付・業務概要・スタンプ数・コメント数 |

---
