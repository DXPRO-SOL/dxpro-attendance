# 勤怠管理 - 画面設計書

---

## 1. 勤怠メイン画面（/attendance-main）

### 概要
社員が日々の打刻を行うメイン画面。リアルタイム時計・GPS打刻・本日の勤怠状況を表示する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| ヘッダー | リアルタイム時計（秒更新）、ユーザー名、部署名 |
| 打刻エリア | 出勤・昼休憩開始・昼休憩終了・退勤 ボタン |
| GPS情報 | 現在地取得状態、承認場所名（圏内/圏外表示） |
| 本日の状況 | 出勤時刻・昼休憩・退勤時刻・実労働時間 |
| 月次サマリー | 出勤日数・総労働時間・残業時間・遅刻/欠勤件数 |

### 項目一覧
| 項目名 | 種別 | 説明 |
|--------|------|------|
| 出勤ボタン | ボタン | GPS検証後に出勤打刻。打刻済みは無効化 |
| 昼休憩開始ボタン | ボタン | 出勤後に有効化。押下で休憩開始時刻を記録 |
| 昼休憩終了ボタン | ボタン | 休憩開始後に有効化。押下で休憩終了時刻を記録 |
| 退勤ボタン | ボタン | 出勤後に有効化。GPS検証後に退勤打刻 |

### ボタン動作
| ボタン | 前提条件 | 押下後の動作 |
|--------|---------|------------|
| 出勤 | 当日未打刻 | GPS取得→POST /checkin→画面更新 |
| 退勤 | 出勤済み・未退勤 | GPS取得→POST /checkout→実労働時間表示 |
| 昼休憩開始 | 出勤済み | POST /start-lunch→時刻表示 |
| 昼休憩終了 | 休憩開始済み | POST /end-lunch→時刻表示 |

### 入力チェック（GPS）
| 条件 | 動作 |
|------|------|
| 位置情報取得不可 | エラーメッセージ表示。打刻ブロック |
| 承認場所から圏外 | エラーメッセージ表示。打刻ブロック（承認場所0件の場合は自由打刻可） |

## 2. 月次一括登録画面（/attendance/bulk-register）

### 概要
対象月の全日を一覧表示し、まとめて出退勤を入力・保存する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| ヘッダー | 年月セレクター |
| 一括入力 | 平日テンプレート（出勤時刻・退勤時刻・昼休憩を一発適用） |
| 日別入力テーブル | 日付・曜日・出勤・昼休憩開始・昼休憩終了・退勤・ステータス・備考 |
| サマリーバー | 出勤/遅刻/早退/欠勤のリアルタイム集計 |
| 保存ボタン | 全行を一括保存 |

### 入力チェック
| 項目 | チェック内容 |
|------|------------|
| 確定済み行 | 編集不可（グレーアウト表示） |
| 承認申請中の月 | 保存操作をブロック（メッセージ表示） |
| 出勤時刻が空の行 | スキップ（エラーなし） |

## 3. 月別勤怠照会画面（/my-monthly-attendance）

### 概要
自分の指定月の勤怠一覧を確認し、月次承認申請を行う。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 月セレクター | 年月選択（前月・翌月ナビ） |
| 勤怠テーブル | 日付・出勤・退勤・昼休憩・実労働時間・ステータス・備考 |
| 月次サマリー | 出勤日数・総労働時間・残業時間・有休日数 |
| 承認申請ボタン | 未申請の場合に表示。申請済みはステータス表示 |

---

## 4. 勤怠編集画面（/edit-attendance/:id）

### 概要
個別勤怠レコードの修正フォーム。

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 日付 | 日付 | 必須 | 対象日（変更不可） |
| 出勤時刻 | 時刻 | 任意 | HH:MM形式 |
| 退勤時刻 | 時刻 | 任意 | HH:MM形式 |
| 昼休憩開始 | 時刻 | 任意 | HH:MM形式 |
| 昼休憩終了 | 時刻 | 任意 | HH:MM形式 |
| ステータス | 選択 | 任意 | 正常/遅刻/早退/欠勤/有休/午前休/午後休/休暇 |
| 備考 | テキスト | 任意 | 自由入力 |

### 入力チェック
| 項目 | チェック |
|------|---------|
| 確定済み勤怠 | 編集不可 |
| 承認申請中の月 | 編集不可 |

---

## 5. 管理者: 月次勤怠一覧（/admin/monthly-attendance）

### 概要
管理者が全社員の月次勤怠状況を確認し承認操作を行う。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | 年月選択、部署フィルター |
| 社員一覧テーブル | 社員名・部署・出勤日数・総労働時間・残業・申請ステータス |
| 承認操作 | 承認申請一覧。承認/差し戻しボタン |
| PDF出力 | 個人ごとの月次勤怠表PDFダウンロード |
