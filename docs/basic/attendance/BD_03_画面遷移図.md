# 勤怠管理 - 画面遷移図

```mermaid
flowchart TD
    Login([ログイン]) --> Main[勤怠メイン
/attendance-main]
    Main --> Bulk[月次一括登録
/attendance/bulk-register]
    Main --> Monthly[月別勤怠照会
/my-monthly-attendance]
    Main --> Edit[勤怠編集
/edit-attendance/:id]
    Monthly --> ApprovalReq[承認申請送信
POST /request-approval]
    ApprovalReq --> Monthly
    Edit --> Monthly
    Bulk --> Main
    Main --> AddNew[勤怠新規追加
/add-attendance]
    AddNew --> Main
    subgraph 管理者
    AdminTop[管理者TOP] --> AdminMonthly[月次勤怠一覧
/admin/monthly-attendance]
    AdminMonthly --> AdminApprove[承認/差し戻し]
    AdminMonthly --> AdminExportPDF[PDF出力]
    end
```

### 遷移条件
| 遷移元 | 遷移先 | 条件 |
|--------|--------|------|
| ログイン画面 | 勤怠メイン | ログイン成功 |
| 勤怠メイン | 月次一括登録 | 「月次入力」ボタン押下 |
| 勤怠メイン | 月別照会 | 「月別確認」ボタン押下 |
| 月別照会 | 承認申請 | 「承認申請」ボタン押下（未申請時のみ） |
| 勤怠メイン | 勤怠編集 | 該当日の「編集」アイコン押下 |
