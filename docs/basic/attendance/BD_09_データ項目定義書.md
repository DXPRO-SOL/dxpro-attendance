# 勤怠管理 - データ項目定義書

---

## 1. 勤怠レコード（Attendance）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 社員のユーザーID | 参照 | 必須 |
| date | 勤務日 | Date | 必須 |
| checkIn | 出勤打刻日時 | Date | 任意 |
| checkOut | 退勤打刻日時 | Date | 任意 |
| lunchStart | 昼休憩開始日時 | Date | 任意 |
| lunchEnd | 昼休憩終了日時 | Date | 任意 |
| workingHours | 実労働時間（時間単位） | 数値 | 自動計算 |
| overtimeHours | 残業時間（時間単位） | 数値 | 自動計算 |
| status | 勤怠ステータス | 区分 | 任意 |
| note | 備考 | 文字列 | 任意 |
| gpsLat | 打刻時GPS緯度 | 数値 | 任意 |
| gpsLng | 打刻時GPS経度 | 数値 | 任意 |
| gpsLocation | 承認場所名 | 文字列 | 任意 |
| confirmed | 管理者承認済みフラグ | 真偽 | 自動 |

### status コード値
| 値 | 説明 |
|----|------|
| 正常 | 所定時間内 |
| 遅刻 | 所定開始時刻後に出勤 |
| 早退 | 実労働時間8時間未満 |
| 欠勤 | 出勤記録なし |
| 有休 | 有給休暇取得日 |
| 午前休 | 午前半休 |
| 午後休 | 午後半休 |
| 休暇 | 病欠・慶弔等 |

---

## 2. 月次承認申請（ApprovalRequest）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 申請者ユーザーID | 参照 | 必須 |
| year | 対象年 | 数値 | 必須 |
| month | 対象月（1〜12） | 数値 | 必須 |
| status | 承認ステータス | 区分 | 自動 |
| returnReason | 差し戻し理由 | 文字列 | 任意 |
| processedBy | 処理した管理者 | 参照 | 自動 |
| processedAt | 処理日時 | Date | 自動 |

### status コード値
| 値 | 説明 |
|----|------|
| pending | 承認待ち |
| approved | 承認済み |
| returned | 差し戻し |
| rejected | 却下 |
