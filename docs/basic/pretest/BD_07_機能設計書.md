# 入社前テスト - 機能設計書

---

## 1. 機能概要
採用候補者がログインなしでオンライン技術テストを受験できる機能。
7ステップ方式で選択式（30問）・記述式（10問）の計40問を出題し、サーバーサイドで自動採点する。

---

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| PRE-01 | テスト受験（認証不要） | 採用候補者 |
| PRE-02 | 自動採点 | システム |
| PRE-03 | 結果保存 | システム |
| PRE-04 | 提出一覧閲覧 | 管理者 |
| PRE-05 | 提出詳細閲覧 | 管理者 |
| PRE-06 | 解説閲覧 | ログイン済みユーザー |

---

## 3. 処理概要

### PRE-01 テスト受験
1. URL: /pretest/:lang に認証なしでアクセス
2. lang に対応した問題セットを lib/pretestQuestions.js から読み込み
3. 7ステップ分割でブラウザに表示（制限時間あり）
4. 全問回答後、POST /pretest/submit で回答を送信

### PRE-02 自動採点
1. computePretestScore(answers, lang) を呼び出し
2. 選択式（Q1〜Q30）: 完全一致で各1点
3. 記述式（Q31〜Q40）: キーワードマッチ数に応じた得点
4. 合格ライン: 40点満点中24点以上（60%）

### PRE-03 結果保存
1. 採点結果・回答内容・受験者名・言語・日時を DB に保存
2. 結果画面にリダイレクトしてスコア・合否を表示
