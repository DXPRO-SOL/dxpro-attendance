# 入社前テスト - データ項目定義書

---

## 1. テスト提出（PretestSubmission）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| lang | 受験言語コード | 文字列 | 必須 |
| candidateName | 受験者名 | 文字列 | 必須 |
| answers | 全問の回答（key-value） | オブジェクト | 必須 |
| score | 合計スコア | 数値 | 自動 |
| total | 満点（40） | 数値 | 自動 |
| perQuestionScores | 問題別スコア | オブジェクト | 自動 |
| passed | 合格フラグ（score ≥ 24） | 真偽 | 自動 |
| submittedAt | 受験日時 | Date | 自動 |

---

## 2. テスト問題定義（lib/pretestQuestions.js）
| 項目名 | 説明 |
|--------|------|
| LANG_TESTS[lang].mc | 選択式問題配列（Q1〜Q30）|
| LANG_TESTS[lang].essay | 記述式問題配列（Q31〜Q40）|
| 選択式.q | 問題文 |
| 選択式.opts | 選択肢配列 |
| 選択式.ans | 正解の選択肢インデックス |
| 選択式.diff | 難易度 |
| 記述式.q | 問題文 |
| 記述式.keywords | 採点キーワード配列 |
