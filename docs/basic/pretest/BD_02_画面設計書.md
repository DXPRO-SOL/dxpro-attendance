# 入社前テスト - 画面設計書

---

## 1. テスト受験画面（/pretest/:lang）

### 概要
認証不要でアクセスできる候補者向けオンラインテスト。7ステップ方式で進む。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| ヘッダー | テスト名・言語・説明文 |
| 進行バー | 現在ステップ / 全ステップ数 |
| 問題エリア | 現在ステップの問題（選択式または記述式） |
| ナビゲーション | 「前へ」「次へ」「送信」ボタン |
| タイマー | 残り時間表示（制限時間あり） |

### 対応言語
| 言語コード | テスト名称 |
|-----------|---------|
| common | 共通 |
| java | Java |
| javascript | JavaScript |
| python | Python |
| php | PHP |
| csharp | C# |
| android | Android |
| swift | Swift |

---

## 2. テスト結果画面（回答送信後）

### 概要
採点完了後に自動遷移。スコアと合否判定を表示。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| スコア表示 | x / 40点（パーセンテージ） |
| 合否判定 | 24点以上: 合格 / 未満: 不合格 |
| セクション別 | 選択式スコア・記述式スコア |

---

## 3. 管理者: 提出一覧（/admin/pretests）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| テーブル | 受験者名・言語・スコア・受験日時 |
| 詳細リンク | 問題別スコアの詳細表示へ遷移 |
| フィルター | 言語・合否・期間 |

---

## 4. 管理者: 提出詳細（/admin/pretest/:id）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 基本情報 | 受験者名・使用言語・合計スコア・受験日時 |
| 問題別結果 | 各問題の回答内容・正誤・得点 |
| 記述式回答 | 回答テキスト・キーワードマッチ数 |

---

## 5. 解説一覧（/pretest/answers/:lang）

### 概要
ログイン済みユーザーが各テストの模範解答を確認できる（採用担当者向け）。
