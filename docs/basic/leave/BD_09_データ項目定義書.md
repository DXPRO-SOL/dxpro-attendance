# 休暇申請 - データ項目定義書

---

## 1. 休暇申請（LeaveRequest）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 申請者ユーザーID | 参照 | 必須 |
| employeeId | 社員ID | 参照 | 任意 |
| name | 申請者氏名（スナップショット） | 文字列 | 必須 |
| department | 所属部署（スナップショット） | 文字列 | 任意 |
| leaveType | 休暇種別 | 区分 | 必須 |
| halfDay | 半休区分 | 区分 | 任意 |
| earlyLeaveTime | 早退時刻 | 文字列 | 任意 |
| startDate | 休暇開始日 | Date | 必須 |
| endDate | 休暇終了日 | Date | 必須 |
| days | 申請日数 | 数値 | 自動 |
| reason | 申請理由 | 文字列 | 任意 |
| status | 申請ステータス | 区分 | 自動 |
| processedBy | 処理した管理者ID | 参照 | 自動 |
| processedAt | 処理日時 | Date | 自動 |
| notes | 却下理由等のメモ | 文字列 | 任意 |

### leaveType コード値
| 値 | 説明 | 残高消費 |
|----|------|---------|
| 有給 | 有給休暇 | 1日（半休: 0.5日） |
| 病欠 | 傷病休暇 | 1日 |
| 慶弔 | 冠婚葬祭 | 1日 |
| その他 | その他 | 1日 |
| 午前休 | 午前半休 | 0.5日 |
| 午後休 | 午後半休 | 0.5日 |
| 早退 | 早退 | 0.5日 |

### status コード値
| 値 | 説明 |
|----|------|
| pending | 承認待ち |
| approved | 承認済み |
| rejected | 却下 |
| canceled | 取消 |

---

## 2. 休暇残高（LeaveBalance）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 社員ユーザーID | 参照 | 必須 |
| paid | 有給残日数 | 数値 | 必須 |
| sick | 病欠残日数 | 数値 | 必須 |
| condolence | 慶弔残日数 | 数値 | 必須 |
| other | その他残日数 | 数値 | 必須 |
| history | 付与・消費の履歴配列 | 配列 | 自動 |
