# 休暇申請 - 画面設計書

---

## 1. 休暇申請フォーム（/leave/apply）

### 概要
社員が休暇の種別・期間・理由を入力して申請する画面。残日数を確認しながら入力できる。

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 休暇種別 | 選択 | 必須 | 有給/病欠/慶弔/その他/午前休/午後休 |
| 半休区分 | 選択 | 条件付 | 午前休/午後休選択時：AM/PM |
| 開始日 | 日付 | 必須 | 休暇開始日 |
| 終了日 | 日付 | 必須 | 休暇終了日（1日の場合は開始日と同じ） |
| 理由 | テキスト | 任意 | 申請理由の自由入力 |
| 残日数表示 | 表示 | — | 有給・病欠・慶弔の現在残日数（参照専用） |
| 次回付与予定バナー | 表示 | — | 次回有給付与予定日・付与日数 |

### 入力チェック
| 項目 | チェック内容 |
|------|------------|
| 開始日 ≤ 終了日 | 終了日が開始日より前の場合エラー |
| 残日数不足 | 申請日数が残日数を超える場合エラー |
| 午前休/午後休 | 期間は1日のみ（開始日=終了日） |

## 2. 早退申請フォーム（/leave/early）

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 早退時刻 | 時刻 | 必須 | HH:MM形式 |
| 理由 | テキスト | 任意 | 早退理由 |
| 有給残日数 | 表示 | — | 有給残高（0.5日単位で消費） |

## 3. 自分の申請一覧（/leave/my-requests）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 残日数サマリー | 有給・病欠・慶弔の残日数カード |
| 申請一覧テーブル | 種別・期間・申請日数・理由・ステータス・操作 |
| 取消ボタン | pending 状態の申請のみ取消可 |

---

## 4. 管理者: 全申請一覧（/admin/leave-requests）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | ステータス・部署・期間でフィルタリング |
| 申請一覧テーブル | 申請者・種別・期間・申請日数・理由・ステータス |
| 承認ボタン | pending 状態の申請に承認ボタン表示 |
| 却下ボタン | pending 状態の申請に却下ボタン表示 |

## 5. 残日数管理（/admin/leave-balance）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 社員一覧テーブル | 社員名・有給残・病欠残・慶弔残・その他残 |
| 付与フォーム | 社員選択・種別・付与日数・メモを入力して付与 |
