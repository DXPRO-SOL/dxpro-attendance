# 会社規定 - 機能設計書

## 1. 機能概要
管理者が就業規則・ハンドブック等の社内規定文書を管理し、社員が閲覧・既読確認できる機能。

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| RUL-01 | 規定作成 | 管理者 |
| RUL-02 | 規定編集・削除 | 管理者 |
| RUL-03 | 規定一覧閲覧 | 全員 |
| RUL-04 | 規定詳細閲覧 | 全員 |
| RUL-05 | 既読確認 | 全員 |
| RUL-06 | 既読状況確認 | 管理者 |

## 3. 処理概要

### RUL-05 既読確認
1. 社員が規定詳細画面で「確認しました」ボタンを押下
2. Rule.confirmations 配列にユーザーIDと確認日時を記録
3. 管理者は既読ユーザー数・未読ユーザー一覧を確認可能
