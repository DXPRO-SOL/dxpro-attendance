# 会社規定 - 画面設計書

---

## 1. 規定一覧（/rules）

### 概要
就業規則・ハンドブック等の社内規定文書を管理する画面。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| カテゴリ | 規定のカテゴリ別タブ |
| 一覧テーブル | 規定名・バージョン・最終更新日・操作 |
| 追加ボタン | 管理者のみ表示 |

---

## 2. 規定詳細・閲覧

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 規定本文 | Markdown/HTMLレンダリング |
| バージョン | 改定履歴一覧 |
| 既読確認 | 社員が「確認しました」を記録可能 |

---

## 3. 規定作成・編集（管理者）

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| タイトル | テキスト | 必須 | 規定名称 |
| カテゴリ | 選択 | 任意 | 就業規則/ハンドブック等 |
| 本文 | テキストエリア | 必須 | Markdown対応 |
| 添付PDF | ファイル | 任意 | PDF添付可 |
