# スキルシート - データ項目定義書

---

## 1. スキルシート（SkillSheet）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| employeeId | 社員ID | 参照 | 必須 |
| userId | ユーザーID | 参照 | 必須 |
| nameKana | 氏名（カナ） | 文字列 | 任意 |
| birthDate | 生年月日 | Date | 任意 |
| gender | 性別 | 文字列 | 任意 |
| nearestStation | 最寄り駅 | 文字列 | 任意 |
| experience | IT経験年数 | 数値 | 任意 |
| selfPR | 自己PR | 文字列 | 任意 |
| skills.languages | プログラミング言語スキル配列 | 配列 | 任意 |
| skills.frameworks | FW・ライブラリスキル配列 | 配列 | 任意 |
| skills.databases | DBスキル配列 | 配列 | 任意 |
| skills.infrastructure | インフラスキル配列 | 配列 | 任意 |
| skills.tools | ツールスキル配列 | 配列 | 任意 |
| qualifications | 資格配列 | 配列 | 任意 |
| workHistory | 職歴配列 | 配列 | 任意 |

### skills 配列要素
| 項目名 | 説明 |
|--------|------|
| name | スキル名 |
| level | 習熟度（1〜5の整数） |

### workHistory 配列要素
| 項目名 | 説明 |
|--------|------|
| projectName | プロジェクト名 |
| startDate | 開始年月 |
| endDate | 終了年月 |
| role | 役割 |
| technologies | 使用技術 |
| processes | 担当工程 |
