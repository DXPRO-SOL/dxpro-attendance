# スキルシート - 機能設計書

---

## 1. 機能概要
社員がプログラミング言語・FW・DB・インフラ等のスキルを習熟度★1〜5で登録し、
資格・職歴も管理できる機能。Excel エクスポートとスキルマップ可視化を提供する。

---

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| SKL-01 | スキルシート登録・更新 | 社員 |
| SKL-02 | 資格登録 | 社員 |
| SKL-03 | 職歴登録 | 社員 |
| SKL-04 | Excelエクスポート | 社員・管理者 |
| SKL-05 | スキルマップ（個人レーダー） | 全員 |
| SKL-06 | スキルマップ（チーム分布） | 管理者 |
| SKL-07 | 他社員スキル閲覧・編集 | 管理者 |

---

## 3. 処理概要

### SKL-04 Excelエクスポート
1. GET /skillsheet/export を呼び出し
2. サーバーが SkillSheet データから xlsx ファイルを生成
3. 2シート構成: 「スキルシート」（基本情報・スキル）・「職歴」
4. Content-Disposition: attachment でダウンロード開始
