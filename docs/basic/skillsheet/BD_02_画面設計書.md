# スキルシート - 画面設計書

---

## 1. スキルシート入力（/skillsheet）

### 概要
社員がスキル・資格・職歴を登録・更新する画面。

### 入力セクション
| セクション | 内容 |
|-----------|------|
| 基本情報 | 氏名（カナ）・生年月日・性別・最寄り駅・IT経験年数・自己PR |
| スキル（言語） | プログラミング言語と習熟度★1〜5 |
| スキル（FW） | フレームワーク・ライブラリと習熟度★1〜5 |
| スキル（DB） | データベースと習熟度★1〜5 |
| スキル（インフラ） | インフラ・クラウドと習熟度★1〜5 |
| スキル（ツール） | 開発ツール・その他と習熟度★1〜5 |
| 資格 | 資格名・取得日 |
| 職歴 | プロジェクト名・期間・役割・使用技術・担当工程 |

---

## 2. スキルマップ（/skillsheet/map）

### 概要
個人のスキルレーダーチャートと、チーム全体のスキル分布を可視化する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 個人レーダーチャート | 5カテゴリ別のスキル平均値レーダー |
| 社員選択 | 比較したい社員を選択して並べて表示 |
| チームバブルチャート | 全社員のスキル総合値分布 |

---

## 3. 管理者: 社員スキルシート（/skillsheet/admin/:employeeId）

### 概要
管理者が任意の社員のスキルシートを閲覧・編集できる。

### ボタン動作
| ボタン | 動作 |
|--------|------|
| 保存 | POST /skillsheet/save（または admin/:id/save）でDB更新 |
| Excelエクスポート | GET /skillsheet/export で xlsx をダウンロード |
