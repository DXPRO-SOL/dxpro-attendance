# ダッシュボード - 画面設計書

---

## 1. ダッシュボード（/dashboard）

### 概要
ログイン後のホーム画面。自分に関連するKPIとAIによる推奨アクションを表示する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| KPIカード | 今月の出勤日数・総労働時間・残業時間・有休残日数 |
| AI推奨 | AIが分析した今日のToDoや業務推奨（管理者設定による） |
| 本日の勤怠 | 出勤状態・打刻時刻（未打刻の場合は促すメッセージ） |
| 承認待ち | 未処理の承認依頼件数（勤怠・目標・ワークフロー等） |
| 最近のお知らせ | 掲示板の最新投稿 |
| スケジュール | 今週のスケジュール一覧 |

### 管理者向け追加表示
| ゾーン | 表示内容 |
|--------|---------|
| 全社員KPI | 本日出勤中社員数・残業アラート件数 |
| 未処理件数 | 勤怠承認待ち・休暇申請待ち件数 |
