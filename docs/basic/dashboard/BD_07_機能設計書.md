# ダッシュボード - 機能設計書

## 1. 機能概要
ログイン後のホーム画面。自分の業務状況のKPIサマリーとAI推奨を表示し、
各機能へのクイックアクセスを提供する。

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| DSH-01 | 個人KPI表示 | 全員 |
| DSH-02 | AI推奨表示 | 全員 |
| DSH-03 | 承認待ち件数表示 | 全員 |
| DSH-04 | 最新掲示板表示 | 全員 |
| DSH-05 | 今週スケジュール表示 | 全員 |
| DSH-06 | 全社員KPI表示 | 管理者 |

## 3. 処理概要

### DSH-01 個人KPI表示
1. GET /dashboard 時に当月の Attendance データを集計
2. 出勤日数・総労働時間・残業時間を表示
3. LeaveBalance から有休残日数を取得して表示

### DSH-02 AI推奨表示
1. 管理者が /ai_home_settings で設定したプロンプトに基づきAIが推奨内容を生成
2. ダッシュボードにウィジェット形式で表示
