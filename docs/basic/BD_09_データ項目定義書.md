# BD-09. データ項目定義書

主要データエンティティの業務的な項目定義を記述する。

---

## 1. 社員（Employee）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| 社員番号 | 社内一意の社員識別番号 | 文字列 | 必須 |
| 氏名 | フルネーム（漢字/カタカナ/英字） | 文字列 | 必須 |
| 部署 | 所属部署名 | 文字列 | 必須 |
| 役職 | 職位名 | 文字列 | 任意 |
| 入社日 | 入社年月日 | 日付 | 必須 |
| 組織ロール | admin/manager/team_leader/employee | 区分 | 任意 |
| メールアドレス | 連絡用・通知送信先 | 文字列 | 任意 |
| 電話番号 | 緊急連絡先 | 文字列 | 任意 |
| ユーザーID | 紐付けアカウントID | 参照 | 必須 |

---

## 2. 勤怠（Attendance）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| ユーザーID | 対象社員 | 参照 | 必須 |
| 日付 | 勤務日 | 日付 | 必須 |
| 出勤時刻 | 出勤打刻時刻 | 日時 | 任意 |
| 退勤時刻 | 退勤打刻時刻 | 日時 | 任意 |
| 昼休憩開始 | 昼休憩開始時刻 | 日時 | 任意 |
| 昼休憩終了 | 昼休憩終了時刻 | 日時 | 任意 |
| 実労働時間 | 自動計算（時間単位） | 数値 | 自動 |
| 残業時間 | 所定時間超過分（時間単位） | 数値 | 自動 |
| ステータス | 正常/遅刻/早退/欠勤/有休等 | 区分 | 任意 |
| 備考 | 自由入力メモ | 文字列 | 任意 |
| GPS緯度/経度 | 打刻時の位置情報 | 数値 | 任意 |
| 確認済みフラグ | 管理者承認済みかどうか | 真偽 | 自動 |

---

## 3. 休暇申請（LeaveRequest）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| ユーザーID | 申請者 | 参照 | 必須 |
| 休暇種別 | 有給/病欠/慶弔/その他/午前休/午後休/早退 | 区分 | 必須 |
| 半休区分 | 午前(AM)/午後(PM) | 区分 | 任意 |
| 開始日 | 休暇開始日 | 日付 | 必須 |
| 終了日 | 休暇終了日 | 日付 | 必須 |
| 申請日数 | 自動計算 | 数値 | 自動 |
| 理由 | 申請理由 | 文字列 | 任意 |
| ステータス | 申請中/承認/却下/取消 | 区分 | 自動 |
| 早退時刻 | 早退の場合の退社時刻 | 文字列 | 任意 |

---

## 4. 目標（Goal）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| タイトル | 目標の名称 | 文字列 | 必須 |
| 説明 | 目標の詳細内容 | 文字列 | 任意 |
| 重要度 | 低/中/高 | 区分 | 任意 |
| 期限 | 達成期限日 | 日付 | 任意 |
| 進捗率 | 0〜100% | 数値 | 任意 |
| アクションプラン | 具体的な行動計画 | 文字列 | 任意 |
| ステータス | 下書き/承認待ち1/承認済1/承認待ち2/完了/差し戻し | 区分 | 自動 |
| 評価グレード | A/B/C/S/D 等（管理者が設定） | 文字列 | 任意 |
| 担当承認者 | 現在の承認担当者 | 参照 | 自動 |

---

## 5. 給与明細（PayrollSlip）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| 給与ランID | 紐付けの給与計算ラン | 参照 | 必須 |
| 社員ID | 対象社員 | 参照 | 必須 |
| 基本給 | 月次基本給与額 | 数値 | 必須 |
| 手当合計 | 各種手当の合計 | 数値 | 自動 |
| 控除合計 | 税・社保等控除合計 | 数値 | 自動 |
| 総支給額 | 基本給＋手当 | 数値 | 自動 |
| 差引支給額 | 総支給額−控除 | 数値 | 自動 |
| ステータス | 下書き/発行済/確定/支払済 | 区分 | 自動 |

---

## 6. 月次承認申請（ApprovalRequest）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| ユーザーID | 申請者 | 参照 | 必須 |
| 対象年 | 対象年度 | 数値 | 必須 |
| 対象月 | 対象月 | 数値 | 必須 |
| ステータス | 承認待ち/承認/差し戻し/却下 | 区分 | 自動 |
| 差し戻し理由 | 差し戻し時のコメント | 文字列 | 任意 |
| 処理者 | 承認/差し戻しを行った管理者 | 参照 | 自動 |
| 処理日時 | 承認等の操作日時 | 日時 | 自動 |

---

## 7. 通知（Notification）

| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| ユーザーID | 受信者 | 参照 | 必須 |
| 種別 | 通知の種類（勤怠承認/目標/休暇等） | 区分 | 必須 |
| タイトル | 通知タイトル | 文字列 | 必須 |
| 本文 | 通知内容 | 文字列 | 任意 |
| リンク先 | クリック時の遷移URL | 文字列 | 任意 |
| 既読フラグ | 既読かどうか | 真偽 | 自動 |
| 受信日時 | 通知作成日時 | 日時 | 自動 |
