## 2. 主要画面レイアウト

### 2-1. 共通レイアウト（認証後全画面）

| エリア | 内容 |
|--------|------|
| ヘッダー | システムロゴ、言語切替、通知ベル（未読件数）、ユーザーメニュー |
| サイドバー | 機能ナビゲーション（PC: 常時表示 / スマホ: ハンバーガーメニュー） |
| メインコンテンツ | 各機能の画面 |
| AI チャットボット | 右下フローティングボタン（全画面共通） |

### 2-2. ログイン画面

| 項目 | 内容 |
|------|------|
| 言語セレクター | 5言語切替（ページリロードなし） |
| 時計 | 現在日時リアルタイム表示 |
| ユーザー名 | 必須テキスト入力 |
| パスワード | 必須・表示/非表示トグル付き |
| ログインボタン | POST /login 送信 |
| エラー表示 | 上部インライン表示（赤帯） |

### 2-3. 勤怠管理トップ

| 項目 | 内容 |
|------|------|
| KPI カード | 今月の出勤日数・総労働時間・残業時間・欠勤日数 |
| 打刻ボタン | 出勤・昼休憩開始・昼休憩終了・退勤（状態に応じてボタン制御） |
| 今日の打刻状況 | 出勤時刻・退勤時刻・労働時間表示 |
| 月次勤怠テーブル | 日付・出退勤時刻・実労働時間・残業・ステータス・備考 |
| 承認申請ボタン | 月次集計完了後に表示 |

## 3. 画面別入力項目・入力チェック

### 3-1. ログイン

| 項目名 | 必須 | チェック内容 |
|--------|------|------------|
| ユーザー名 | 必須 | 空文字NG |
| パスワード | 必須 | 空文字NG、サーバー照合 |

### 3-2. パスワード変更

| 項目名 | 必須 | チェック内容 |
|--------|------|------------|
| 現在のパスワード | 必須 | サーバー照合 |
| 新しいパスワード | 必須 | 8文字以上 |
| 確認用パスワード | 必須 | 新しいパスワードと一致 |

### 3-3. 休暇申請

| 項目名 | 必須 | チェック内容 |
|--------|------|------------|
| 休暇種別 | 必須 | 選択必須 |
| 開始日 | 必須 | 日付形式 |
| 終了日 | 必須 | 開始日以降 |
| 申請日数 | 自動計算 | 有給の場合は残日数以内 |
| 理由 | 任意 | — |

### 3-4. 目標作成

| 項目名 | 必須 | チェック内容 |
|--------|------|------------|
| タイトル | 必須 | 最大200文字 |
| 説明 | 任意 | — |
| 重要度 | 必須 | 低/中/高 |
| 期限 | 任意 | 日付形式 |
| アクションプラン | 任意 | — |

### 3-5. 残業申請

| 項目名 | 必須 | チェック内容 |
|--------|------|------------|
| 申請区分 | 必須 | 事前/事後 |
| 日付 | 必須 | 日付形式 |
| 開始時刻 | 必須 | HH:MM形式 |
| 終了時刻 | 必須 | HH:MM形式、開始より後 |
| 残業種別 | 必須 | 通常/休日出勤/深夜/その他 |
| 理由 | 必須 | — |

## 4. 主要ボタン動作一覧

| 画面 | ボタン名 | 動作 |
|------|---------|------|
| 勤怠トップ | 出勤 | GPS座標取得→打刻登録 |
| 勤怠トップ | 退勤 | GPS座標取得→退勤登録 |
| 月次勤怠 | 承認申請 | 管理者へ承認リクエスト送信＋通知 |
| 目標一覧 | 新規作成 | 目標作成画面へ遷移 |
| 目標詳細 | 提出 | 状態を「承認待ち（1次）」へ変更＋承認者へ通知＋メール |
| 目標詳細 | 承認 | 次ステップへ進める（管理者/マネージャー操作） |
| 休暇申請 | 申請する | 申請登録＋管理者へ通知 |
| 掲示板 | いいね | いいね数カウントアップ |
| 通知一覧 | 全件既読 | 全通知を既読状態に変更 |
| 管理者承認 | 承認 | 勤怠確定＋社員へ通知＋メール |
| 管理者承認 | 差し戻し | 勤怠確定取消し＋社員へ通知 |
