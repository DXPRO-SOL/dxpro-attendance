# 認証 - データ項目定義書

---

## 1. ユーザー（User）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| username | ログインユーザー名（一意） | 文字列 | 必須 |
| password | bcryptハッシュ済みパスワード | 文字列 | 必須 |
| displayName | 表示名 | 文字列 | 任意 |
| role | システムロール | 区分 | 必須 |
| orgRole | 組織ロール | 区分 | 任意 |
| preferredLang | 優先表示言語 | 区分 | 任意 |
| chatStatus | チャット在席状態 | 区分 | 自動 |
| lastLogin | 最終ログイン日時 | Date | 自動 |
| isActive | アカウント有効フラグ | 真偽 | 必須 |

### role コード値
| 値 | 説明 |
|----|------|
| admin | 管理者（全機能） |
| manager | マネージャー |
| team_leader | チームリーダー |
| employee | 一般社員 |
| test_user | テストユーザー（閲覧専用） |

### preferredLang コード値
| 値 | 言語 |
|----|------|
| ja | 日本語 |
| en | 英語 |
| vi | ベトナム語 |
| ko | 韓国語 |
| zh | 中国語（簡体） |
