# BD-04. 帳票設計書

---

## 1. 出力帳票一覧

| # | 帳票名 | 形式 | 出力操作者 | 配布先 |
|---|--------|------|----------|--------|
| 1 | 月次勤怠表（管理者用） | PDF | 管理者 | 経理・税理士事務所（メール添付） |
| 2 | 月次勤怠表（社員用） | PDF | 社員・管理者 | 本人確認・印刷用 |
| 3 | 給与明細 | PDF | 管理者 | 社員 |

---

## 2. 月次勤怠表

### 出力条件

| 条件 | 内容 |
|------|------|
| 出力タイミング | 月次承認完了後、または管理者が任意のタイミングで出力 |
| 出力操作者 | 管理者（全社員分）/ 社員（自分の分） |
| ファイル名 | 勤怠表_{氏名}_{年}年{月}月.pdf |
| 用紙サイズ | A4 横向き |

### 出力項目

| 区分 | 項目 |
|------|------|
| ヘッダー | 社員名・社員番号・部署・対象年月・出力日 |
| 日別明細 | 日付（曜日）・出勤時刻・退勤時刻・昼休憩・実労働時間・残業時間・ステータス・備考 |
| 集計サマリー | 出勤日数・欠勤日数・有休使用日数・総労働時間・残業時間合計 |

---

## 3. 給与明細

### 出力条件

| 条件 | 内容 |
|------|------|
| 出力タイミング | 管理者が給与ランを「発行」操作した後 |
| 出力操作者 | 管理者 |
| ファイル名 | 給与明細_{氏名}_{年}年{月}月.pdf |
| 用紙サイズ | A4 縦向き |

### 出力項目

| 区分 | 項目 |
|------|------|
| ヘッダー | 社員名・社員番号・部署・支給対象月 |
| 支給 | 基本給・各手当・通勤費（非課税/課税）・残業手当 |
| 控除 | 所得税・各控除項目 |
| 差引支給額 | 手取り額（総支給額 − 控除合計） |
