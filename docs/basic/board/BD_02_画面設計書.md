# 掲示板 - 画面設計書

---

## 1. 掲示板一覧（/board）

### 概要
会社全体の掲示板。投稿・コメント・いいね・ピン留めができる。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | カテゴリ・キーワード検索 |
| ピン留め投稿 | 管理者がピン留めした重要投稿を上部に固定表示 |
| 投稿一覧 | 投稿者・タイトル・本文（抜粋）・カテゴリ・いいね数・コメント数・日時 |
| 新規投稿ボタン | 管理者のみ表示 |

---

## 2. 投稿詳細（/board/:id）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 投稿本文 | 投稿者・カテゴリ・日時・本文（Markdown対応） |
| 添付ファイル | ダウンロードリンク |
| いいねボタン | 押下でトグル。件数表示 |
| コメントリスト | コメント者・本文・日時 |
| コメント入力 | テキストエリア + 送信ボタン |

---

## 3. 投稿フォーム（管理者）

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| タイトル | テキスト | 必須 | 投稿のタイトル |
| カテゴリ | 選択 | 任意 | お知らせ/一般/業務連絡等 |
| 本文 | テキストエリア | 必須 | Markdown対応 |
| 添付ファイル | ファイル | 任意 | 複数可 |
| ピン留め | チェックボックス | 任意 | 一覧上部に固定表示 |
