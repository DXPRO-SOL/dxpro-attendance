# 掲示板 - 機能設計書

## 1. 機能概要
全社員が閲覧できる社内掲示板。管理者が投稿し、社員がコメント・いいね・ファイル閲覧ができる。

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| BRD-01 | 投稿作成 | 管理者 |
| BRD-02 | 投稿編集・削除 | 管理者 |
| BRD-03 | ピン留め | 管理者 |
| BRD-04 | 投稿一覧閲覧 | 全員 |
| BRD-05 | 投稿詳細閲覧 | 全員 |
| BRD-06 | コメント投稿 | 全員 |
| BRD-07 | いいね | 全員 |
| BRD-08 | ファイル添付 | 管理者 |

## 3. 処理概要

### BRD-01 投稿作成
1. 管理者がタイトル・本文・カテゴリ・添付ファイルを入力
2. POST /board で投稿を作成
3. ピン留め設定の場合は一覧上部に固定表示
