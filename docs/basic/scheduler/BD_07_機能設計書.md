# スケジューラ - 機能設計書

## 1. 機能概要
個人およびチームのスケジュールをカレンダー形式で管理する。
参加者への招待・承認/辞退応答・公開範囲設定ができる。

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| SCH-01 | イベント作成 | 全員 |
| SCH-02 | イベント編集・削除 | 作成者・管理者 |
| SCH-03 | カレンダー表示 | 全員 |
| SCH-04 | 参加者招待 | イベント作成者 |
| SCH-05 | 招待承認/辞退 | 招待された社員 |
| SCH-06 | チームスケジュール表示 | 全員 |

## 3. 処理概要

### SCH-04 参加者招待
1. イベント作成時に参加者を選択
2. Schedule.attendees 配列に追加（status: pending）
3. 招待された社員へ通知

### SCH-05 招待承認/辞退
1. 招待された社員が承認 → attendees[].status: accepted
2. 辞退 → attendees[].status: declined
