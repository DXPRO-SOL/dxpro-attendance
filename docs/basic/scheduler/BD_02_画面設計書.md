# スケジューラ - 画面設計書

---

## 1. スケジュール一覧・カレンダー（/schedule）

### 概要
個人・チームのスケジュールをカレンダー形式で管理する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 月次カレンダー | 各日のイベントをカラーバーで表示 |
| 週次ビュー | タイムライン形式 |
| フィルター | 個人/チーム/全体の表示切り替え |
| イベント追加ボタン | 新規イベント作成フォームへ |

---

## 2. イベント作成・編集

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| タイトル | テキスト | 必須 | イベント名称 |
| 種別 | 選択 | 必須 | 会議/イベント/その他 |
| 開始日時 | 日時 | 必須 | イベント開始 |
| 終了日時 | 日時 | 必須 | イベント終了 |
| 場所 | テキスト | 任意 | 開催場所 |
| 説明 | テキストエリア | 任意 | イベント詳細 |
| 参加者 | 選択（複数） | 任意 | 招待するユーザー |
| 公開設定 | 選択 | 任意 | private/public |
| スコープ | 選択 | 任意 | personal/team |
