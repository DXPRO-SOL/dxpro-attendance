# 管理者機能 - 画面設計書

---

## 1. 管理者ダッシュボード（/admin）

### 概要
管理者専用のメニュー集約画面。各管理機能へのリンクと概要数値を表示する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| 承認待ちサマリー | 月次勤怠承認待ち件数・休暇申請承認待ち件数 |
| 管理メニュー | 勤怠管理・社員管理・給与管理・休暇管理へのカードリンク |
| 最近の操作 | 直近の管理操作ログ |

---

## 2. 月次勤怠管理（/admin/monthly-attendance）

### 概要
全社員の月次勤怠サマリーを一覧表示し承認・差し戻し操作を行う。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | 年月・部署 |
| 社員テーブル | 社員名・部署・出勤日数・総労働時間・残業・申請ステータス |
| 承認操作 | 各社員行に承認/差し戻しボタン（pending時のみ） |
| 個別PDF | 社員ごとの月次勤怠表PDFダウンロード |

---

## 3. 承認場所管理（/admin/locations または /locations）

### 概要
GPS打刻を許可する承認場所（会社・テレワーク拠点等）を管理する。

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| 場所名 | テキスト | 必須 | 承認場所の名称 |
| 緯度 | 数値 | 必須 | 場所の緯度 |
| 経度 | 数値 | 必須 | 場所の経度 |
| 許容半径（m） | 数値 | 必須 | この半径内からのみ打刻を許可 |
| 有効フラグ | 真偽 | 必須 | 無効化しても削除せず保持 |
| 対象ユーザー | 選択（複数） | 任意 | 空の場合は全員に適用 |

---

## 4. ユーザー管理（/admin/users）

### 概要
ログインユーザーの作成・権限変更・パスワードリセットを行う。

### 項目一覧
| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| ユーザー名 | テキスト | 必須 | ログインID（一意） |
| 表示名 | テキスト | 任意 | 画面表示名 |
| パスワード | パスワード | 必須 | 作成時のみ設定 |
| ロール | 選択 | 必須 | admin/manager/team_leader/employee/test_user |
| 有効/無効 | 真偽 | 必須 | アカウント有効化フラグ |

---

## 5. 監査ログ（/auditlog）

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| フィルター | 操作種別・ユーザー・日時範囲 |
| ログテーブル | 日時・ユーザー・操作種別・対象・結果・IPアドレス |
| CSV出力 | フィルタ条件で絞り込んだログをCSV出力 |
