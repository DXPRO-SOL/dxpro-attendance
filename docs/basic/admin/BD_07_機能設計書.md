# 管理者機能 - 機能設計書

---

## 1. 機能概要
システム管理者が全社員の勤怠・申請の承認・ユーザー管理・GPS場所管理・監査ログ確認等の
管理業務を行うための機能群。

---

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| ADM-01 | 月次勤怠一覧・承認 | 管理者 |
| ADM-02 | 月次勤怠差し戻し | 管理者 |
| ADM-03 | 勤怠PDF一括出力 | 管理者 |
| ADM-04 | ユーザー作成・編集 | 管理者 |
| ADM-05 | ロール変更 | 管理者 |

| ADM-07 | 承認場所登録・編集 | 管理者 |
| ADM-08 | 承認場所有効/無効 | 管理者 |
| ADM-09 | 監査ログ閲覧 | 管理者 |
| ADM-10 | 監査ログCSV出力 | 管理者 |
| ADM-11 | AI設定（ホーム推奨等） | 管理者 |

---

## 3. 処理概要

### ADM-01 月次勤怠承認
1. 社員からの ApprovalRequest（status: pending）を一覧表示
2. 管理者が承認 → 対象月の全 Attendance レコードを confirmed: true に更新
3. ApprovalRequest.status → approved
4. 社員へ通知・メール送信

### ADM-07 承認場所登録
1. 場所名・緯度・経度・半径・対象ユーザーを入力
2. ApprovedLocation レコードを作成
3. 打刻時のGPS検証で自動参照される
