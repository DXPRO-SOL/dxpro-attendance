# BD-07. 機能設計書

各機能の業務概要・処理概要を記述する。

---

## 1. 勤怠管理

### 機能概要
社員が日々の出退勤を記録し、月次で管理者が承認するまでの一連の業務を管理する。

### 主要処理
| 処理 | 概要 |
|------|------|
| GPS打刻 | スマートフォンの位置情報を取得し、登録された承認場所（会社・テレワーク等）の半径内であれば打刻を受け付ける |
| 手動打刻 | 管理者が代理で日付・時刻を指定して勤怠を登録する |
| 月次集計 | 月の出勤日数・総労働時間・残業時間・欠勤日数を自動集計する |
| 承認フロー | 社員が月次勤怠を確認後、管理者へ承認依頼を送信。管理者は承認/差し戻しを行う |
| 承認完了処理 | 承認後、当月全勤怠を「確認済み」に変更。以降は社員による修正不可 |

## 2. 休暇管理

### 機能概要
社員が休暇申請を行い、管理者が承認することで残高を自動管理する。

### 主要処理
| 処理 | 概要 |
|------|------|
| 申請 | 休暇種別・期間・理由を入力して申請。申請日数は自動計算 |
| 残高チェック | 有給・午前/午後休は申請時に残日数をチェック。不足の場合はエラー |
| 承認 | 管理者が承認すると残高が差し引かれ、対象日の勤怠ステータスが更新される |
| 却下 | 管理者が却下すると残高は変わらず、申請は却下状態になる |
| 通知・メール | 申請・承認・却下の各タイミングで関係者へ通知とメールを送信 |

---

## 3. 目標管理（MBO）

### 機能概要
社員が個人目標を設定し、2段階承認（マネージャー→管理者）で確定する。

### 主要処理
| 処理 | 概要 |
|------|------|
| 目標作成 | タイトル・説明・重要度・期限・アクションプランを入力して下書き保存 |
| 進捗更新 | 0〜100% の進捗率をいつでも更新可能 |
| 1次承認申請 | マネージャーへ承認依頼送信。マネージャーが承認/差し戻し |
| 2次承認申請 | 1次承認後、管理者へ最終承認依頼送信 |
| グレード設定 | 最終承認時に管理者が評価グレードを設定可能 |

---

## 4. 給与管理

### 機能概要
管理者が給与計算・明細発行を行い、社員が自分の明細を確認する。

### 主要処理
| 処理 | 概要 |
|------|------|
| 給与ラン作成 | 対象月・対象社員を選択して給与計算ランを作成 |
| 明細作成 | 基本給・各手当・控除を入力して明細を作成 |
| 発行 | 明細を「発行」状態にすると社員から閲覧可能になる。メール通知送信 |
| PDF出力 | 社員・管理者ともに PDF でダウンロード可能 |
| CSV出力 | 社員・管理者ともに CSV でダウンロード可能 |

---

## 5. 日報管理

### 機能概要
社員が日次の業務報告を記録し、管理者が AI 要約で把握する。

### 主要処理
| 処理 | 概要 |
|------|------|
| 日報作成 | 業務内容・時間・翌日計画を入力して日次保存 |
| AI 要約 | 管理者が週次・月次で全社員日報の AI 要約を取得 |
| メール送付 | AI 要約を任意のメールアドレスに送付 |

---

## 6. スキルシート管理

### 機能概要
社員がスキル・資格・職歴を登録し、Excel エクスポートや管理者向けスキルマップを提供する。

### 主要処理
| 処理 | 概要 |
|------|------|
| 情報登録 | プログラミング言語・FW・DB・インフラ・ツールのカテゴリ別に習熟度（★1〜5）を設定 |
| 資格登録 | 資格名・取得日を登録 |
| 職歴登録 | プロジェクト名・期間・役割・技術・担当工程を登録 |
| Excel出力 | 2シート形式のスキルシートを Excel でダウンロード |
| スキルマップ | 管理者が全社員のスキル分布をレーダーチャートで確認 |

---

## 7. ワークフロー

### 機能概要
管理者が定義した任意の申請フォームを使って、多段階承認の電子申請を行う。

### 主要処理
| 処理 | 概要 |
|------|------|
| フォーム定義 | 管理者がフォーム項目（テキスト/数値/日付/選択肢）と承認ステップを定義 |
| 申請作成 | 社員がフォームに入力して申請を送信 |
| 承認操作 | 各ステップの承認者が承認/差し戻しを行う |
| 連番採番 | 申請ごとに一意のシリアルナンバーを自動採番 |

---

## 8. 通知システム

### 機能概要
業務イベントをリアルタイムで関係者に通知する。

### 主要処理
| 処理 | 概要 |
|------|------|
| リアルタイム通知 | Socket.IO でブラウザにリアルタイムプッシュ配信 |
| バッジ更新 | ヘッダーの通知ベルに未読件数をリアルタイム表示 |
| 既読管理 | 通知単位・全件一括で既読に変更可能 |
| メール連携 | 通知に合わせてメール送信（設定時） |

---

## 9. 入社前テスト

### 機能概要
採用候補者がログインなしでオンラインテストを受験できる。

### 主要処理
| 処理 | 概要 |
|------|------|
| テスト受験 | 多肢選択・記述の7ステップ方式。制限時間あり |
| 採点 | サーバーサイドで自動採点（スコア0〜100） |
| 結果保存 | 受験者名・スコア・回答内容を DB に保存 |
| 結果閲覧（管理者） | 管理者が受験者一覧とスコアを確認 |
