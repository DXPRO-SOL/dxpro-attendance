# 目標管理 - 画面遷移図

```mermaid
flowchart TD
    Login([ログイン]) --> GoalList[目標一覧
/goals]
    GoalList --> Add[目標作成
/goals/add]
    GoalList --> Edit[目標編集
/goals/edit/:id]
    GoalList --> Detail[目標詳細
/goals/detail/:id]
    Add -->|下書き保存/申請| GoalList
    Edit -->|更新| Detail
    Detail -->|1次申請| Detail
    Detail -->|評価入力| Evaluate[評価フォーム
/goals/evaluate/:id]
    Evaluate -->|2次申請| Detail
    Detail -->|差し戻し| Detail
    subgraph 承認者
    ApprovalList[承認待ち一覧
/goals/approval] -->|承認/差し戻し| Detail
    end
    subgraph 管理者
    Report[目標レポート
/goals/report]
    end
```

### 遷移条件
| 遷移元 | 遷移先 | 条件 |
|--------|--------|------|
| 目標一覧 | 目標作成 | 「新規作成」ボタン押下 |
| 目標一覧 | 目標詳細 | カード押下 |
| 目標詳細 | 評価フォーム | approved1 状態 + 作成者本人 |
| 目標詳細 | 承認操作 | pending1/pending2 状態 + 承認者/管理者 |
