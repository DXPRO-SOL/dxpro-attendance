# 目標管理 - データ項目定義書

---

## 1. 目標（Goal）
| 項目名 | 説明 | 型 | 必須 |
|--------|------|-----|------|
| userId | 作成者ユーザーID | 参照 | 必須 |
| title | 目標タイトル | 文字列 | 必須 |
| description | 目標の詳細説明 | 文字列 | 任意 |
| goalLevel | 重要度（低/中/高） | 区分 | 任意 |
| dueDate | 達成期限 | Date | 任意 |
| progress | 進捗率（0〜100） | 数値 | 任意 |
| actionPlan | アクションプラン | 文字列 | 任意 |
| grade | 評価グレード | 文字列 | 任意 |
| status | 承認ステータス | 区分 | 自動 |
| currentApprover | 現在の承認担当者ID | 参照 | 自動 |
| history | 操作履歴配列 | 配列 | 自動 |

### status コード値
| 値 | 説明 |
|----|------|
| draft | 下書き |
| pending1 | 1次承認待ち（マネージャー） |
| approved1 | 1次承認済み |
| pending2 | 2次承認待ち（管理者） |
| completed | 最終承認完了 |
| rejected | 差し戻し/却下 |

### history 配列要素
| 項目名 | 説明 |
|--------|------|
| action | 操作種別（create/submit1/approve1/reject1/submit2/approve2/reject2/evaluate） |
| userId | 操作者ID |
| comment | 操作コメント・理由 |
| timestamp | 操作日時 |
