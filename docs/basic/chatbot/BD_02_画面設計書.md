# AIチャットボット - 画面設計書

---

## 1. チャットボットウィジェット（全ページ埋め込み）

### 概要
ページ右下に常時表示されるAIチャットウィジェット。
社内ナレッジ（勤怠・休暇・給与等）に関する質問に回答する。

### 画面レイアウト
| ゾーン | 表示内容 |
|--------|---------|
| ウィジェットボタン | 右下固定のチャットボットアイコン（未読バッジ付き） |
| チャットパネル | クリックで展開。会話履歴・入力欄・送信ボタン |
| メッセージ | ユーザー発話（右）・AI応答（左）の吹き出し形式 |
| タイピングインジケーター | AI応答中は「...」表示 |

---

## 2. チャットボット設定（管理者）（/chatbot/settings または /ai_home_settings）

### 概要
AIチャットボットの動作設定（システムプロンプト・対応業務等）を管理者が変更する。

### 項目一覧
| 項目名 | 種別 | 説明 |
|--------|------|------|
| システムプロンプト | テキストエリア | AIの役割・回答ルールを定義 |
| ダッシュボード推奨設定 | テキストエリア | ホーム画面のAI推奨内容の設定 |
| 有効/無効 | 真偽 | チャットボットの表示を制御 |
