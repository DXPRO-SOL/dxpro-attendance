# AIチャットボット - 機能設計書

## 1. 機能概要
全ページに埋め込まれたAIチャットウィジェット。社員が勤怠・休暇・給与等に関する
質問を自然言語で入力すると、AIが社内データを参照して回答する。

## 2. 機能一覧
| 機能ID | 機能名 | 利用者 |
|--------|--------|--------|
| BOT-01 | 質問・回答チャット | 全員 |
| BOT-02 | 会話履歴表示 | 全員 |
| BOT-03 | チャットボット設定 | 管理者 |
| BOT-04 | ダッシュボードAI推奨 | 管理者 |

## 3. 処理概要

### BOT-01 質問・回答
1. 社員がチャットパネルに質問を入力して送信
2. POST /chatbot/api/chat を呼び出し
3. AIがシステムプロンプト（管理者設定）と社内データを参照
4. 回答をストリーミングまたは一括でレスポンス
5. チャットパネルにメッセージとして表示
