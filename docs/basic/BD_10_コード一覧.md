# BD-10. コード一覧

---

## 1. ユーザーロール（User.role / Employee.orgRole）

| コード | 表示名 | 説明 |
|--------|--------|------|
| admin | 管理者 | 全機能アクセス可能。給与・承認・設定操作可能 |
| manager | マネージャー | 部門管理・目標1次承認 |
| team_leader | チームリーダー | チーム勤怠確認 |
| employee | 一般社員 | 自分のデータのみ操作可能 |
| test_user | テストユーザー | 閲覧専用（書き込み操作ブロック） |

## 2. 勤怠ステータス（Attendance.status）

| コード | 表示名 | 説明 |
|--------|--------|------|
| 正常 | 正常 | 所定時間内の出退勤 |
| 遅刻 | 遅刻 | 所定開始時刻後に出勤 |
| 早退 | 早退 | 所定終了時刻前に退勤 |
| 欠勤 | 欠勤 | 無断欠勤または承認外欠勤 |
| 有休 | 有給休暇 | 有給取得日 |
| 午前休 | 午前休暇 | 午前半休取得日 |
| 午後休 | 午後休暇 | 午後半休取得日 |
| 休暇 | 休暇（有給以外） | 病欠・慶弔等 |

|--------|--------|---------|

| 慶弔 | 慶弔休暇 | なし |
| その他 | その他 | なし |
| 午前休 | 午前半休 | あり（0.5日） |
| 午後休 | 午後半休 | あり（0.5日） |

## 4. 休暇申請ステータス（LeaveRequest.status）

| コード | 表示名 | 説明 |
|--------|--------|------|
| pending | 申請中 | 管理者未処理 |
| approved | 承認済 | 管理者承認済み |
| rejected | 却下 | 管理者により却下 |
| canceled | 取消 | 申請者が取消 |

| approved1 | 1次承認済み | マネージャー承認済み |

| rejected | 差し戻し | 承認者により差し戻し |

## 8. 残業申請区分・ステータス

### 申請区分（OvertimeRequest.type）
| コード | 説明 |
|--------|------|
| pre | 事前申請（残業前に申請） |
| post | 事後申請（残業後に申請） |

### 残業種別（OvertimeRequest.overtimeType）
| コード | 説明 |
|--------|------|
| 通常残業 | 平日の法定内・外残業 |
| 休日出勤 | 法定休日・所定休日の出勤 |
| 深夜残業 | 22:00〜5:00 の労働 |
| その他 | 上記以外 |

### 申請ステータス
| コード | 説明 |
|--------|------|
| pending | 承認待ち |
| approved | 承認済み |
| rejected | 却下 |
| canceled | 取消 |
