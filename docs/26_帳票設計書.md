# 26. 帳票設計書

html-pdf ライブラリ（^3.0.1）を使用して HTML → PDF 変換。

---

## 1. 帳票一覧

| # | 帳票名 | 出力形式 | 生成エンドポイント |
|---|--------|---------|-----------------|
| 1 | 月次勤怠表 | PDF | GET /admin/export-pdf/:userId/:year/:month |
| 2 | 月次勤怠表（社員用） | PDF | GET /attendance/export-pdf/:year/:month |
| 3 | 給与明細 | PDF | GET /hr/payroll/:employeeId/pdf |

---

## 2. 月次勤怠表

### 2-1. 基本情報ヘッダー

| 項目 | 値 |
|------|----|
| 社員名 | Employee.name |
| 社員番号 | Employee.employeeId |
| 部署 | Employee.department |
| 対象年月 | year 年 month 月 |
| 出力日 | 出力時の現在日時 |

### 2-2. 日別明細テーブル

| 列 | 内容 |
|----|------|
| 日付 | YYYY/MM/DD（曜日） |
| 出勤時刻 | checkIn（HH:mm） |
| 退勤時刻 | checkOut（HH:mm） |
| 昼休憩 | lunchStart〜lunchEnd |
| 実労働時間 | workingHours（h） |
| 残業時間 | overtimeHours（h） |
| ステータス | 正常 / 遅刻 / 早退 / 欠勤 / 有休 等 |
| 備考 | notes |

### 2-3. 月次集計サマリー

| 項目 | 計算式 |
|------|--------|
| 出勤日数 | status が「正常」「遅刻」「早退」の件数 |
| 欠勤日数 | status が「欠勤」の件数 |
| 有休使用日数 | status が「有休」「午前休」「午後休」の件数 |
| 総労働時間 | workingHours の合計 |
| 残業時間合計 | overtimeHours の合計 |

### 2-4. 出力設定

```js
pdf.create(html, {
  format: "A4",
  orientation: "landscape",
  border: "10mm"
}).toStream(callback)
```

レスポンスヘッダー:
```
Content-Type: application/pdf
Content-Disposition: attachment; filename="勤怠表_YYYY年MM月.pdf"
```

## 3. 給与明細

### 3-1. ヘッダー情報

| 項目 | 値 |
|------|----|
| 社員名 | Employee.name |
| 社員番号 | Employee.employeeId |
| 部署 | Employee.department |
| 支給対象月 | PayrollRun.periodFrom〜periodTo |

### 3-2. 支給・控除テーブル

| 支給 | 控除 |
|------|------|
| 基本給 | 所得税 |
| 手当（allowances[]） | 各控除（deductions[]） |
| 通勤費（非課税） | — |
| 通勤費（課税） | — |
| 残業手当 | — |

### 3-3. 差引支給額（手取り）

`net = gross - sum(deductions) - incomeTax`

### 3-4. 出力設定

同上（A4 縦向き）
